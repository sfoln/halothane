Uninstall Halothane completely

These steps remove the app, privileged helper, launch services, configuration, preferences, caches, log, and installer receipt.

First, resume anything Halothane paused. Open the Halothane menu and choose Resume for every paused item. A stopped process remains stopped if its supervisor exits; if one remains paused after removal, restart your Mac.

Recommended: inspect and run the removal script

  1. Download the Halothane uninstaller.
  2. Open the downloaded file in a text editor and inspect the commands.
  3. Open Terminal and run:
cd ~/Downloads
sudo bash ./uninstall-halothane.sh

Administrator approval is required because Halothane installs system-wide components.

What the script removes

  • /Applications/Halothane.app
  • /Library/LaunchDaemons/com.sfoln.Halothane.Helper.plist
  • /Library/LaunchAgents/com.sfoln.Halothane.GUI.plist
  • /Library/Application Support/Halothane/
  • /var/log/halothane-helper.log
  • Halothane preferences, caches, saved state, and application-support data in each local user's Library
  • The com.sfoln.Halothane.pkg installer receipt

Verify removal

After the script reports completion, Halothane should no longer appear in Applications or the menu bar. Restart the Mac if a menu item remains or a previously paused process has not resumed. For help, email support@halothane.app.
